Latest Projects

Path visualizer

Shortest Path Finding visualization tool using Python pygame and A* shortest path algorithm. Used Manhattan distance formula to calculate the distance between the current node to the last node and updated neighbor nodes according to that and then reconstructed the path using backtracking on the dictionary containing from which node we came to the current node.

The application is Open Source and available on Github

YelpCamp

A website where users can register and login to add new campgrounds or rate and review the already added ones.

Frontend - HTML, CSS, BootStrap5, Ejs, CSS, Flexbox, JavaScript

Backend - Node js, Express, Session, Cookies, Passport js, Mongoose, joi, MapBox API

Database - MongoDB

Used Express Mongo Sanitize and joi to prevent the website from Mongo Injection and Cross Site Scripting.

The application is Open Source and available on Github
